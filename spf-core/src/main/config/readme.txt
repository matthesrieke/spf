Sensor Platform Framework
=============================================

Installation
------------
Extract the contents of the archive to a folder of your choice.


Running the SPFramework
------------
The release includes executables for Windows and Linux 32/64 bit operating systems.
Just run the suitable executable. If you are using a Linux distribution and you
downloaded the zip-archive you need to set executive rights for the scripts.
